HTML Elements

Element Description Block, Inline or N/A Container or Empty
<!-- --> HTML Comments tag is used to insert comments in the source code but not displayed in the browsers. N/A Container
<!DOCTYPE> It's not an HTML tag but it's the very first thing in HTML document, so that the browser knows what type of document to expect. N/A Empty
<a> It is a hyperlink which is used to link from one page to another. Inline Container
<article> It is a self-contained content which can contain entire blog posts, news articles and similar content. Block Container
